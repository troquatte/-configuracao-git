<h1>GIT</h1>
<p>Alguns comandos que me ajudam no Git</p>

<h2>Git Reset</h2>
<p>Reseta todos as alterações dos arquivos</p>
<code>
  git reset --hard origin/master -
  git clean -f -
  git pull
</code>

<h2>Git Checkout muda a Brantch ( Master, release ou dev )</h2>
<p>Reseta todos as alterações dos arquivos</p>
<code>
  git checkout {Brantch}
</code>
